# Aplikasi Anti-Bully

Aplikasi ini bertujuan untuk membantu mendeteksi dan mencegah perilaku bullying di lingkungan sekolah.

## Fitur Utama
- Manajemen data siswa, subjek, dan relasi
- Input dan edit data melalui antarmuka
- Analisis relasi antar siswa
- Backup dan restore database

## Cara Instalasi
1. Pastikan Python sudah terinstall di komputer Anda.
2. Install semua dependensi dengan menjalankan:
   ```
   pip install -r Requirements.txt
   ```
3. Jalankan aplikasi dengan perintah:
   ```
   streamlit run Aplikasi.py
   ```

## Struktur Folder
- `Aplikasi.py` : Source code utama aplikasi
- `Database.db` : Database utama
- `Database_backup.db` : Backup database
- `Requirements.txt` : Daftar dependensi Python
- `Run.bat` : Script untuk menjalankan aplikasi
- `Installer Lengkap.bat` : Script instalasi otomatis
- `Requirements installer.bat` : Script instalasi dependensi
- `Diagram Alur.txt` : Diagram alur aplikasi
- `Konsep.txt` : Penjelasan konsep aplikasi
- `Menu Tree.txt` : Struktur menu aplikasi
- `TODO.md` : Daftar tugas pengembangan

## Catatan
- Pastikan semua dependensi di `Requirements.txt` sudah terinstall.
- Jika ada error terkait modul, tambahkan modul tersebut ke `Requirements.txt` dan install ulang.
